# サクピタ 開発ログ

## プロダクト概要

**サクピタ**（Saku-Pita）は、営業チームが顧客との商談日程を効率よく調整・予約できるWebアプリ。
Googleカレンダーと連携し、チームメンバー全員の空き時間をリアルタイムで確認しながら、顧客情報の入力・メール送信・スプレッドシート記録までワンストップで行える。

- **URL**: https://jun-digiman.github.io/sakupita/
- **技術スタック**: 純粋な HTML / CSS / JavaScript（フレームワークなし）
- **外部API**: Google Calendar API / Gmail API / Google Sheets API / Google Drive API
- **認証**: Google Identity Services (GIS) OAuth 2.0

---

## 開発の経緯・目的

- 既存の日程調整ツール（Calendly等）は外部向けで、社内の複数メンバーのカレンダーを一括で見て調整する機能がない
- 顧客への確認メール送信・スプレッドシートへの記録など、商談調整に必要な一連の作業を1つのツールに集約したかった
- Googleアカウントだけで完結するシンプルな設計にこだわった

## 実装した機能一覧

### 🔐 認証・セッション管理

- **Googleサインイン**: Google Identity Services (GIS) の `requestAccessToken` を使用
- **トークンのlocalStorage保存**: アクセストークンを期限付きでlocalStorageに保存し、リロード時に復元。毎回の再認証を不要にした
- **サイレント自動更新**: トークン期限の5分前にバックグラウンドで自動更新するタイマーを実装。タブを開いている限り再認証画面が出ない
- **再接続オーバーレイ**: 万が一トークンが切れた場合、カレンダー表示を維持しつつ「再接続」ボタンを表示
- **サインアウト**: ヘッダーにサインアウトボタン（SVGアイコン）を設置。サインアウト時はlocalStorageを全クリアしてサインイン画面へ戻る

### 📅 カレンダー表示

- **週表示**: 月〜日の7列グリッドで時間帯別に空き/予約済みスロットを表示
- **前週・次週ナビゲーション**
- **当日ハイライト**: 今日の日付を紺色の丸に白文字で表示
- **土日のカラー分け**: 土曜=青系、日曜=赤系
- **busyDataキャッシュ**: Googleカレンダーの空き時間データを2時間TTLでlocalStorageにキャッシュ。毎回APIを叩かずに高速表示
- **リアルタイム反映**: 予約完了後、カレンダーグリッドを即時更新

### 👥 メンバー管理

- **メンバー追加・削除**: 名前とGmailアドレスで追加
- **連携状態表示**: 「連携済み」「未連携」バッジ
- **カラー割り当て**: メンバーごとに自動でカラーを割り当て、カレンダー上で識別しやすく
- **URLからメンバー追加**: `calendar.app.google` のURLを貼り付けてメンバー追加を試みる機能（CORSプロキシ3種を順番に試行）

### 📝 予約フォーム

**入力項目**（コメント以外すべて必須）:
| 項目 | バリデーション |
|------|--------------|
| 会社名 | 必須 |
| 担当者名 | 必須・自動スペース補完 |
| メールアドレス | 必須・形式チェック |
| 電話番号 | 必須・ハイフン形式チェック |
| 部署 | 必須 |
| 役職 | 必須 |
| コメント | 任意 |

**UXこだわりポイント**:
- フォームを開いたとき会社名フィールドに自動フォーカス
- 全フィールドにグレー文字のプレースホルダー例を表示
- タブ移動時に電話番号を自動ハイフン変換（全角・ハイフンなし入力にも対応）
- タブ移動時に氏名の全角スペース・スペースなしを自動で半角スペースに統一
- 電話番号は市外局番・携帯・フリーダイヤルで区切り方を自動調整

### 📞 電話番号フォーマット (`formatPhone`)

```
11桁（090/080/070等）: XXX-XXXX-XXXX
10桁（03/06等）: XX-XXXX-XXXX
10桁（0120/0570等）: XXXX-XXX-XXX
10桁（その他）: XXX-XXX-XXXX
```

### 🏷 日本語名前自動分割 (`splitJapaneseName`)

- 800件以上の日本語姓辞書（`JP_SURNAMES`）を搭載
- 最長一致アルゴリズムで苗字を正確に判定しスペース挿入
- 沖縄系姓（比嘉・金城・玉城・知念等）も含む
- 5文字姓（勅使河原等）から1文字姓まで対応

### 📧 メール送信

- **顧客・担当者への確認メール**: Gmail APIで自動送信
- **バウンス検知**: メール送信後5秒待機し、Gmailの受信箱をmailer-daemon/postmasterで検索。バウンスメールが届いていた場合はアラートを表示
- **送信ステータスのカレンダー記録**:
  - ✅ 送信成功 → カレンダーイベントの説明欄に記録
  - ⚠️ バウンス検知 → イベントタイトルに「⚠️ メール未達」を追加、説明欄にも記録

### 📆 Googleカレンダーイベント作成

- **Google Meet自動生成**: 予約時にMeet URLを自動発行し、メール本文に記載
- **イベント説明欄のフォーマット**:
```
【会社名】株式会社〇〇
【顧客名】山田 太郎様
【部署名・役職名】営業部 部長
【電話番号】090-1234-5678
【メールアドレス】yamada@example.com
【コメント】（任意）

✅ 確認メール送信済み
送信先: yamada@example.com
```

### 📊 Googleスプレッドシート連携

- **自動作成**: 初回予約時にGoogleドライブへ「サクピタ 予約データ」シートを自動生成
- **列構成**: 企業名 / 担当者名 / 部署 / 役職 / 電話番号 / メールアドレス / メール送信日 / 商談日 / コメント
- シートIDはlocalStorageに保存し、次回以降は同じシートに追記

## 技術的こだわりポイント

### ⚡ 高速化

- **Discovery Docs廃止**: `gapi.client.init`でdiscovery docsを読み込まず、`gapi.client.request()`の直接呼び出しに統一 → 約600ms短縮
- **preconnect / dns-prefetch**: `<head>`にGoogleのAPIドメインへの事前接続ヒントを追加
- **busyDataキャッシュ**: 週単位・メンバー別にキャッシュし、同じ週のナビゲーション時はAPIを再呼び出ししない

### 🔄 リロード耐性

- アクセストークンをlocalStorageに保存し、`gapi.client.setToken()`で復元
- ページリロード後もサインイン画面を経由せずにカレンダーが即表示される
- 55分ごとのサイレント自動更新でセッションを維持

### 🛡 バリデーション設計

- フロントエンドのみで完結する入力チェック
- エラー時は該当フィールドをハイライト + アラートで具体的なフォーマットを案内
